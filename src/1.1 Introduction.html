Вступление в мир JavaScript

JavaScript является одним из наиболее распространенных и востребованных языков программирования в мире веб-разработки. В этом разделе мы погружаемся в мир JavaScript, изучая его историю, ключевые возможности и преимущества, а также его важную роль в современной веб-разработке.

Что такое JavaScript?

JavaScript (JS) - это высокоуровневый язык программирования для создания интерактивных веб-страниц. Он используется для манипуляции HTML-элементами, обработки событий, отправки запросов к серверу без перезагрузки страницы (AJAX), создания анимаций и многое другое.

JavaScript это не Java

JavaScript (JS) и Java - это два различных языка программирования с разными историями, целями и областями применения.

Роль JavaScript в веб-разработке

JavaScript играет решающую роль в создании интерактивных и динамических веб-приложений. Он используется для добавления функциональности на стороне клиента, обеспечивая взаимодействие пользователя с веб-страницей без необходимости перезагрузки страницы.

JavaScript также широко используется на стороне сервера с помощью платформы Node.js, что позволяет создавать мощные и масштабируемые веб-приложения.

Однако JavaScript может выполняться не только в браузере, но и на сервере или на любом другом устройстве, которое имеет специальную программу, называющуюся "движком" JavaScript и браузера есть собственный движок

Разные движки имеют разные "кодовые имена". Например:

• V8 – в Chrome, Opera и Edge.
• SpiderMonkey – в Firefox.
• Ещё есть "Chakra" для IE, "JavaScriptCore", "Nitro" и "SquirrelFish" для Safari и т.д.

Как работают движки?

Движки сложны. Но основы понять легко.

1. Движок (встроенный, если это браузер) читает («парсит») текст скрипта.
2. Затем он преобразует («компилирует») скрипт в машинный язык.
3. После этого машинный код запускается и работает достаточно быстро.

Движок применяет оптимизации на каждом этапе. Он даже просматривает скомпилированный скрипт во время его работы, анализируя проходящие через него данные, и применяет оптимизации к машинному коду, полагаясь на полученные знания. В результате скрипты работают очень быстро.

JavaScript может выполнять следующие роли:

• Изменение контента и стилей: Добавление, изменение и удаление HTML-элементов на странице, а также изменение их стилей.

• Реакция на действия пользователя:Обработка событий, таких как щелчки мыши, нажатия клавиш, отправка форм, и изменение содержимого страницы в ответ на эти действия.

• Взаимодействие с сервером: Отправка асинхронных запросов на сервер (AJAX), загрузка данных без перезагрузки страницы, отправка форм, загрузка файлов и другие операции связанные с сетью.

• Управление куками: Установка, получение и удаление куков на стороне клиента для хранения информации между сессиями.

• Работа с браузером: Управление размером и положением окна браузера, создание всплывающих окон и другие действия, связанные с интерфейсом браузера.

• Анимация и визуализация: Создание анимаций, визуальных эффектов и прочих интерактивных элементов, делающих страницу более привлекательной для пользователя.

В чём особенность JavaScript

Как миниму, три сильные стороны JavaScript

  • Полная интеграция с HTML/CSS
  • Простые вещи делаются просто
  • Поддердивается всеми основными браузерами и включён по умолчанию

    • JavaScript – это единственная браузерная технология, сочетающая в себе все эти три вещи.

    Хотя, конечно, JavaScript позволяет делать приложения не только в браузерах, но и на сервере, на мобильных устройствах и т.п.
