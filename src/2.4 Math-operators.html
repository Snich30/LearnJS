Математические операторы

Математические операторы в JavaScript позволяют выполнять различные арифметические операции над числами. Это фундаментальная часть языка, которая необходима для выполнения большинства вычислений и манипуляций с данными. В этой главе мы рассмотрим основные математические операторы, а также приведем примеры их использования.

Основные математические операторы

1. Сложение (+) Используется для сложения двух чисел.

1234 
let a = 5;
let b = 3;
let sum = a + b;
console.log(sum); // 8

3. Умножение (*)

Используется для умножения двух чисел.

1234 
let a = 5;
let b = 3;
let product = a * b;
console.log(product); // 15

4. Деление (/)

Используется для деления одного числа на другое.

1234 
let a = 6;
let b = 3;
let quotient = a / b;
console.log(quotient); // 2

5. Остаток от деления (%)

Используется для получения остатка от деления одного числа на другое.

1234 
let a = 7;
let b = 3;
let remainder = a / b;
console.log(remainder); // 1

6. Возведение в степень (**)

Используется для возведения числа в степень.В школьной математике мы записываем это как ab

1234 
let a = 2;
let b = 3;
let power = a / b;
console.log(power); // 8

Унарные операторы

Унарным называется оператор, который применяется к одному операнду. Например, оператор унарный минус "-" меняет знак числа на противоположный:

1. Инкремент (++)

Увеличивает значение переменной на единицу.

123 
let a = 5;
a++;
console.log(a); // 6

2. Дикремент (--)

Уменьшает значение переменной на единицу.

123 
let a = 5;
a--;
console.log(a); // 4

Важно!

Инкремент/декремент можно применить только к переменной. Попытка использовать его на значении, типа 5++, приведёт к ошибке.

Операторы ++ и -- могут быть расположены не только после, но и до переменной.

  • Когда оператор идёт после переменной — это «постфиксная форма»: counter++.
  • «Префиксная форма» — это когда оператор идёт перед переменной: ++counter.

    • Обе эти инструкции делают одно и то же: увеличивают counter на 1.

    Приоритет операторов

    Важно понимать, что в JavaScript существует определенный порядок выполнения операций, называемый приоритетом операторов. Операции с более высоким приоритетом выполняются раньше операций с более низким приоритетом.

    12 
    let a = 5 + 3 * 2;
console.log(a); // 11, потому что умножение выполняется раньше сложения

    Чтобы изменить порядок выполнения операций, можно использовать круглые скобки:

    12 
    let a = (5 + 3) * 2;
console.log(a); // 16, потому что сначала выполняется сложение в скобках

    Сложение строк при помощи бинарного +

    Обычно при помощи плюса + складывают числа.

    Но если бинарный оператор + применить к строкам, то он их объединяет в одну:

    12 
    let a = "моя" + "строка";
alert(a); // моястрока

    Обратите внимание, если хотя бы один операнд является строкой, то второй будет также преобразован в строку.

    Например:

    12 
    alert("1" + 2); // "12" 
alert(2 + "1"); // "21"

    Как видите, не важно, первый или второй операнд является строкой.

    Вот пример посложнее:

    1 
    alert(2+2+"1"); // // будет "41", а не "221"

    Здесь операторы работают один за другим. Первый + складывает два числа и возвращает 4, затем следующий + объединяет результат со строкой, производя действие 4 + '1' = '41' .

    Сложение и преобразование строк — это особенность бинарного плюса +. Другие арифметические операторы работают только с числами и всегда преобразуют операнды в числа.

    Смешанные операторы

    Математические операторы могут быть использованы в сочетании с операторами присваивания для упрощения кода.

    1. Сложение и присваивание (+=)

    123 
    let a = 5;
a+=3;// эквавалентно a = a + 3
console.log(a); // 8

    2. Вычитание и присваивание (-=)

    123 
    let a = 5;
a-=3;// эквавалентно a = a - 3
console.log(a); // 2

    3. Умножение и присваивание (*=)

    123 
    let a = 5;
a*=3;// эквавалентно a = a * 3
console.log(a); // 15

    4. Деление и присваивание (/=)

    123 
    let a = 6;
a/=3;// эквавалентно a = a / 3
console.log(a); // 2

    5. Остаток от деления и присваивание (%=)

    123 
    let a = 7;
a%=3;// эквавалентно a = a % 3
console.log(a); // 1

    Побитовые операторы

    Побитовые операторы работают с 32-разрядными целыми числами (при необходимости приводят к ним), на уровне их внутреннего двоичного представления.

    Эти операторы не являются чем-то специфичным для JavaScript, они поддерживаются в большинстве языков программирования.

    Поддерживаются следующие побитовые операторы:

  • AND(и) ( & ).
  • OR(или) ( | ).
  • XOR(побитовое исключающее или) (ˆ).
  • NOT(не) (~).
  • LEFT SHIFT(левый сдвиг) (<<).
  • RIGHT SHIFT(правый сдвиг) (>>).

    • Они используются редко, когда возникает необходимость оперировать с числами на очень низком (побитовом) уровне. В ближайшем времени они нам не понадобятся, так как веб-разработчики редко к ним прибегают, хотя в некоторых сферах (например, в криптографии) они полезны.
